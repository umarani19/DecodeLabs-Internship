# PROJECT 1: TO-DO LIST
## Overview
This project is a simple To-Do List made using Python.It helps users add,view,and delete tasks easily.The project is useful for managing daily tasks and learning basic Python programming.
## Objective
The main aim of this project is to learn Python basics by creating a simple task management application.It also helps in improving coding and logical thinking skills.
## Tools Used
- Python
- VS Code
- Command Prompt
## Conclusion
By doing this project,I learned how to use Python concepts like lists,loops,functions,and conditions.I also learned how to take user input and create a simple working application.
